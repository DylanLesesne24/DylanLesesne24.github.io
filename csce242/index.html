Assignment 1

Assignment 1 tests the student's knowledge on basic html by creating a simple page with a title, paragraphs, tables, lists, pictures, and functional links.

Assignment 2

Assignment 2 expands upon assignment 1 by bringing in all of those skills learned and adding css on top of it which allows for a website to be brought to life.

Assignment 3

Assignment 3 dives deeper into css by introducing "flex". This allows the student to make a website more organized and presentable.

Assignment 4

Assignment 4 tests the students flex skills by replicating a more advanced website using HTML5 and higher level CSS.

Assignment 5

Assignment 5 tests all of the skills given in the previous assignments by tasking them with recreating a USC website.

Assignment 6

Assignment 6 tasks the student with expanding upon their main page by adding a neatly designed projects section.

Assignment 7

Assignment 7 tasks the students with working on javascript skills on top of the already learned css and html.

Assignment 8

Assignment 8 expands upon previous javascript skills and tests the student on a deeper level.

Assignment 9

Assignment 9 introduces loops in JavaScript, allowing the student to create more dynamic and efficient code.

Assignment 10

Assignment 10 introduces JavaScript functions, helping students organize and reuse code effectively.

Assignment 11

Assignment 11 builds on JavaScript by introducing event handling, allowing students to make their web pages interactive in response to user actions.
